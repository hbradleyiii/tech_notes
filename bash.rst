History Expansion
=================


Refering to Previous Commands
-----------------------------

.. code:: bash

    $ # Refer to the previous command
    $ !!
    $ # -or-
    $ !-1

    $ # History with Substitution:
    $ echo 'hi'
    hi
    $ !!:s/hi/hello/ # Prints 'hello
    hello
